# Tea Dojo AI Hackathon Challenges

This repository contains groundwork documentation for 8 AI hackathon challenge statements designed for **Tea Dojo**, a bubble tea shop in Ang Mo Kio, Singapore.

## About Tea Dojo

Tea Dojo is a bubble tea destination offering a variety of beverages including classic milk teas, fruit teas, specialty coffee drinks, and traditional Asian beverages. Beyond their retail storefront, Tea Dojo provides event catering services through bubble tea live stations and push cart services.

**Location:** 713A Ang Mo Kio Ave 6, #01-14, Singapore 561713  
**Website:** [https://teadojosg.com/](https://teadojosg.com/)

## Getting Started

Each challenge folder contains:

- `README.md` - Detailed problem statement and background
- `GROUNDWORK.md` - Technical groundwork, suggested approaches, and resources
- `data/` - Sample data structures and mock datasets (where applicable)
- `resources/` - Additional resources, references, and documentation

## Evaluation Criteria

Solutions will be evaluated based on:

| Criterion | Weight | Description |
|-----------|--------|-------------|
| **Innovation** | 25% | Creativity and novelty of the AI solution |
| **Feasibility** | 25% | Technical viability for a small business |
| **Business Impact** | 25% | Potential to drive revenue or reduce costs |
| **User Experience** | 15% | Intuitive design for staff and customers |
| **Scalability** | 10% | Potential for broader F&B industry adoption |
